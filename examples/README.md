# Examples

Runnable examples for **Intor Translator**.

---

### Node.js (TypeScript)

Runnable examples executed in a Node.js environment using `tsx`,
demonstrating programmatic usage of the translator APIs.

```bash
tsx examples/basic.ts
```

```bash
tsx examples/handler.ts
```

```bash
tsx examples/hook.ts
```

```bash
tsx examples/render-rich-message.ts
```

### HTML

```bash
npm run examples:html
```
